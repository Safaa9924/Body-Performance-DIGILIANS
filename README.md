# Body-Performance-DIGILIANS

Physical Performance Analysis (Machine Learning Project)

🤖 Project Overview:
This project applies machine learning techniques to analyze physical performance data and predict:
Fitness classification (Classification task)
Broad jump performance (Regression task)

🎯 Objectives:
Build predictive models for fitness performance
Compare multiple machine learning algorithms
Identify key features affecting physical performance
🛠️ Tools & Technologies:
Python (Pandas, NumPy, Scikit-learn)
Machine Learning Algorithms:
Decision Tree, KNN , SVM ,Linear Regression, Neural Networks (ANN).

⚙️ Methodology:
Data preprocessing and feature engineering
Model training using multiple algorithms
Performance evaluation using:
Accuracy (classification)
RMSE, MSE, R² (regression)
Model comparison to select best-performing approach

🔍 My Role:
As part of the team, I contributed to:
Performing Exploratory Data Analysis (EDA):
Outlier Detection

EDA Summary
Implementing and analyzing the Decision Tree model:
Classification and regression tasks
Model evaluation (accuracy, RMSE, R²)
Participating in the final presentation

📊 Key Results:
Decision Tree achieved solid performance in both:
Classification (~69% accuracy)
Regression (~77% R²)
Neural Networks and SVM showed higher overall performance

💡 Business Value:
Enables fitness prediction without physical testing
Supports personalized training programs
Can be applied in health, sports, and fitness industries
